Проблема

Создание экзаменационных билетов занимает много времени преподавателей.
Копировать-вставить и самостоятельно располагать вопросы набивает оскомину.

Решение

Мы решили автоматизировать процесс создания билетов с использованием технологий
встроенных в офисный пакет корпорации Microsoft
и других открытых по лицензии GNU LGPL технологиях.

Используемые технологии

  • С++ версии 17, стандартная библиотека классов и функций
  • Кроссплатформенный фреймворк Qt
  • Microsoft ActiveX
